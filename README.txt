This zip file has wrods for design and summary of the project, 
java file for executing
project2.java is a java file for the project.

Compile project2.java with command 
"javac project2.java"
Run the program with command
"java project2 3 3"
The two numbers indicate the number of doctor and patient for the simulation.
The first number is for the number of docotor, and the second number is for the number of patient
The number of doctors should be between 0 and 3 and the number of patients should be between 0 and 30.
